# simple-calculator-python-

The Simple Calculator is a basic Python-based application designed to perform fundamental arithmetic operations such as addition, subtraction, multiplication, and division. It allows the user to select an operation from a menu, enter two numeric values, and instantly view the calculated result.

This project is console-based and user-friendly, making it ideal for beginners to understand core Python concepts. It demonstrates the use of input/output statements, conditional logic, loops, and functions. Proper validation ensures smooth execution, such as handling division by zero.
